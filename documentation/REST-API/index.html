This can only be used by an Administrator.

Parameters

id (required)
Form Parameter — Identifier of the harvest definition.
name (required)
Form Parameter — Name of the harvest definition. This string is used in the GUI.
type (required)
Form Parameter — Type of harvesting method.
url (required)
Form Parameter — Url of the endpoint of the repository.
ns (required)
Form Parameter — Metadata namespace of the metadata records.
adminEmail (optional)
Form Parameter — Technical contact info of the repository.
isPendingByDefault (required)
Form Parameter — Whether the harvested records should be marked as Pending for approval by default. default: off
isCheckingBrokenLink (required)
Form Parameter — Whether the harvested records should be checked for broken links by default. default: off
isCheckingInvalid (optional)
Form Parameter — Whether the harvested records should be marked when they do not comply to the specified application profile. default: off
invalidApplProf (optional)
Form Parameter — URI of the application profile to check if isCheckingInvalid is true.
xsl (optional)
Form Parameter — XSLT code that will be applied to harvested records before import.

String
