Dans ce type de recherche, deux choix s'offrent à vous :

  • Choix d'une catégorie: dans la première case, entrez un mot afin de faire apparaitre une liste de catégories dans diverses classifications. Vous pourrez sélectionner la catégorie désirée. 
  • Choix d'une classification: dans la seconde liste, sélectionnez directement la classification afin d'y naviguer depuis la racine.

Dans le fil d'Ariane vous pouvez explorer les sous-catégories de la classification sélectionnée afin d'affiner votre recherche.
Pour inclure des catégories équivalentes provenant d’une autre classification, cochez la case et spécifiez votre choix.

Cliquez sur un résultat pour afficher sa fiche descriptive.

Dans la fiche, cliquez sur pour accéder aux données liées (Linked data).
